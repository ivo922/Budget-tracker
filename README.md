# Budget Tracker

Local-first mobile budget tracker built with Expo and React Native Paper.

## Quick start

```bash
cd ~/Projects/budget-tracker
npm install
npx expo start
```

Press `i` for iOS Simulator or `a` for Android Emulator.

## Features (v1)

- Multiple accounts with computed balances
- Income, expense, and transfer transactions
- Hierarchical categories (parent + subcategory)
- Period views: today, week, month, year
- Analytics with spending-by-category charts
- All data stored locally in SQLite (no login, no cloud)
